# Vidéo de la bannière d'accueil

Déposez ici un fichier nommé exactement `hero.mp4` (format MP4, sans son de préférence puisqu'elle joue automatiquement en muet, quelques secondes en boucle, idéalement en paysage) : un extrait de concert, une ambiance de salle, des billets scannés à l'entrée, etc.

Tant qu'aucun fichier `hero.mp4` n'est présent, la bannière affiche automatiquement un fond dégradé animé (aucune erreur visible, juste un rendu plus sobre).

Dès que vous déposez `hero.mp4` ici et le poussez sur GitHub, la vidéo remplace le fond animé automatiquement — aucune autre modification de code n'est nécessaire.

Astuce : gardez le fichier léger (quelques Mo, 10-20 secondes) pour que la page reste rapide à charger.
